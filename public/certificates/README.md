# Certificate Files Directory

This directory contains all certificate PDF/image files for the portfolio.

## File Structure

Place your certificate files with these exact names:

### Required Certificate Files:
- `oracle-ai-foundations.pdf` - Oracle Cloud Infrastructure 2025 Certified AI Foundations Associate
- `palo-alto-foundation.pdf` - Cybersecurity Foundation (Palo Alto Networks)
- `forage-analyst.pdf` - Cybersecurity Analyst Job Simulation (Forage)
- `nptel-cryptography.pdf` - Cryptography & Network Security (NPTEL)
- `null-vadodara.pdf` - Diving Deep in Cyber Security (Null Vadodara)
- `ec-council-cloud.pdf` - Cloud Computing (EC-Council)

## File Formats Supported:
- ✅ PDF (recommended for certificates)
- ✅ JPG/JPEG (good for scanned certificates)
- ✅ PNG (good for digital certificates)

## File Size Recommendations:
- Keep files under 5MB for optimal loading
- For PDFs: 1-3MB is ideal
- For images: 500KB-2MB is ideal

## How to Add Your Certificates:

1. **Scan or download** your certificates
2. **Name them exactly** as listed above
3. **Place them** in this `public/certificates/` directory
4. **Test the download** by clicking "Download Certificate" on your portfolio

## Notes:
- The portfolio will automatically detect and display download buttons for certificates that have files
- If a certificate file is missing, the download button will not appear
- All certificate files are publicly accessible at `/certificates/filename.pdf`
