# Certificate Files Placeholder

This directory is ready for your certificate files.

## To Add Your Certificates:

1. Replace this file with your actual certificate PDFs/images
2. Use these exact filenames:
   - oracle-ai-foundations.pdf
   - palo-alto-foundation.pdf
   - forage-analyst.pdf
   - nptel-cryptography.pdf
   - null-vadodara.pdf
   - ec-council-cloud.pdf

3. Supported formats: PDF, JPG, JPEG, PNG

## Example File Structure:
```
public/certificates/
├── oracle-ai-foundations.pdf
├── palo-alto-foundation.pdf
├── forage-analyst.pdf
├── nptel-cryptography.pdf
├── null-vadodara.pdf
├── ec-council-cloud.pdf
└── README.md (this file)
```

Once you add the actual certificate files, visitors to your portfolio will be able to download them by clicking the "Download Certificate" button in the certificate modal.
